# AI API Assignment

Python programs integrating 6 different AI APIs.

## Setup
1. Clone this repo
2. Run: pip install -r requirements.txt
3. Create a .env file with your API keys (see .env.example)

## Files
- openai_chat.py — OpenAI GPT integration
- groq_chat.py — Groq LLaMA integration
- ollama_chat.py — Ollama local model integration
- huggingface_chat.py — Hugging Face model integration
- gemini_chat.py — Google Gemini integration
- cohere_chat.py — Cohere integration

## Usage
Run any file: python filename.py
